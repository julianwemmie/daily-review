# Introduction

Amber is a beautiful, opinionated spaced repetition app with AI-powered grading and FSRS scheduling. Create cards manually, import from Anki or Mochi, or generate them automatically with the CLI and Claude Code plugin.

## Why Amber?

Spaced repetition is the most effective way to retain knowledge long-term, but existing tools are either ugly and overwhelming (Anki) or lack AI integration (Mochi). Amber is designed to be simple, beautiful, and smart.

- **AI-powered grading** — answer in your own words, not rote memorization
- **Optimal scheduling** — FSRS algorithm maximizes retention with minimal review time
- **Multiple ways to add cards** — create manually, import from Anki/Mochi, or auto-generate via CLI

## How it works

1. Add cards — create them in the app, import a deck, or use the CLI/plugin to generate them
2. Review due cards each day — the AI grader evaluates your responses
3. Rate yourself (Again / Hard / Good / Easy) and FSRS schedules the next review
4. Cards resurface at optimal intervals so you retain what you learn

## Quick start

Head to [app.amber.cards](https://app.amber.cards) to create an account and start reviewing. If you want to set up the CLI and Claude Code plugin for automatic card generation, see [Installation](/docs/installation).
