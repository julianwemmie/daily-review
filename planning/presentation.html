How it works

The Claude plugin continuously monitors your conversations and automatically extracts flashcard-worthy knowledge. You don't do anything.

It looks for generalizable, flashcard-friendly insights and uploads them without you thinking about it.

~
$ claude
// You code normally...
// Behind the scenes:
> Analyzing conversation... ✓
> Found 3 flashcard-worthy insights
> Uploaded to Amber ✓

Learning happens in Claude.
Retention happens in Amber.
